## 과제

### 제공 자료

1. **Terraform 설정** (`main.tf`) — 현재 인프라 구성
2. **메트릭 데이터** (`metrics/metrics.json`) — 30일간 CloudWatch 메트릭
3. **비용 리포트** (`cost_report.json`) — 6개월 비용 히스토리

### 기대 산출물
